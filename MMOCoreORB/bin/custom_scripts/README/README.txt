-----------------------------
How to restore custom scripts:
-----------------------------
1. Add the contents of allobjects.txt to /bin/scripts/object/allobjects.lua
2. Add the contents of serverobjects.txt to /bin/scripts/object/serverobjects.lua 
3. Add the contents of schematics.txt to /bin/scripts/managers/crafting/schematics.lua
4. Add the contents of items.txt to /bin/scripts/loot/items.lua
5. Add the contents of groups.txt to /bin/scripts/loot/groups.lua
6. Add the contents of screenplays.txt to /bin/scripts/screenplays/screenplays.lua
